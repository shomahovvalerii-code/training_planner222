# Movie Library (Личная кинотека)

**Автор:** Дмитрий Кузнецов

## Описание

**Movie Library** — десктопное приложение с графическим интерфейсом для ведения личной коллекции фильмов. Позволяет добавлять фильмы с указанием названия, жанра, года выпуска и рейтинга, просматривать список в удобной таблице, фильтровать по жанру и году, а также сохранять все данные в JSON-файле. Приложение автоматически загружает сохранённую библиотеку при запуске.

## Функциональность

- ➕ Добавление фильма с полями: название, жанр, год выпуска, рейтинг.
- 📋 Отображение всех фильмов в таблице с сортировкой (сначала новые, затем по алфавиту).
- 🔍 Фильтрация по жанру (выбор из выпадающего списка предустановленных жанров).
- 📅 Фильтрация по году выпуска (диапазон: «от» и «до»).
- ❌ Удаление выбранного фильма.
- 💾 Автоматическое сохранение данных в файл `movie_library.json`.
- 🔄 Загрузка данных из JSON при запуске.
- ✅ Валидация ввода:
  - Год — целое число в диапазоне 1888–2100.
  - Рейтинг — число от 0.0 до 10.0.
  - Название и жанр не могут быть пустыми.

- Все модули из стандартной библиотеки (`tkinter`, `json`, `os`)
